## 📋 Обо мне

Я QA-инженер с практическим опытом функционального, регрессионного и интеграционного тестирования веб- и мобильных приложений. Работаю в краудтестинговой компании **Testbirds** (проект **ivi**).

- 🔍 Провожу ручное и автоматизированное тестирование (Java / Kotlin).
- 🛠️ Работаю с API (REST, SOAP), базами данных (SQL, NoSQL) и инструментами перехвата трафика.
- 🎓 Имею профильное образование и сертификаты.
